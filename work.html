Project 01

The Daily Todo

Full-Stack Task Management App

01 Context

The real flex is the CI/CD pipeline. I wanted that green checkmark so badly I built an entire app around it. Every push, every automated deployment, every successful build felt like I was finally doing software "the right way." The newspaper design was just window dressing for the main event: watching GitHub Actions work its magic. The features reminders, dark mode, themes were necessary, but the pipeline was the love story. I needed a reason to containerize something real. Now I have a todo list that deploys itself.

02 Architecture

Frontend Static HTML/CSS/JS served via Nginx
Backend Node.js + Express.js REST API
Database MongoDB with persistent volumes
Containers Docker Compose orchestration
CI/CD GitHub Actions automated pipeline

03 Tech Stack

Docker Docker Compose Node.js Express.js MongoDB Nginx GitHub Actions

Project 02

Sales Tracker

Offline-First Field Sales App

01 Context

This one's personal. My dad works at small distribution business, and I used to help out when I can. Watching him scroll through spreadsheets on his phone at dinner squinting at rows, trying to remember who still owed money felt wrong. I downloaded every sales app I could find. They were either bloated with features he'd never touch or locked behind subscriptions that made no sense for a one-man operation. So I built what I wished existed something that answers three questions fast Who owes me? What did I sell? What's my report? No cloud, no login, no monthly rent. Just a tool that respects your data and your time. The offline-first choice? Because all data stored on device no sync needed. This is software for real work, not venture scale fantasies.

02 Philosophy

No Cloud Your data lives on your device. No sync issues. No leaks.
No Accounts Download and start. No sign-up walls.
No Subs You own the tool. No monthly rent.

03 Capabilities

Offline First Zero latency. Works in the middle of nowhere.
Inventory Auto deducts stock. Warns you when low.
Ledger Tracks outstanding balances per shop automatically.
Invoicing Generates clean, professional PDF invoices.
Payments QR code generation for instant UPI payments.

04 Tech Stack

React 19 Vite Dexie.js (IndexedDB) Capacitor Tailwind CSS

Project 03

Life Calendar Wallpaper

Daily Year Calendar Generator

01 Overview

I found thelifecalendar.com one night during an existential crisis, as one does. The concept hit me like a truck your entire life as a grid of weeks. Terrifying and motivating at the same time.But the wallpaper it generated? It fought with my clock. No padding options, no color tweaks, no way to make it mine.So I built my own. One that lets me add padding for my lock screen widgets, pick colors that match my vibe, and updates itself automatically via a shortcut it ran every day. The auth and rate limiting came later turns out other people wanted their own existential dread wallpapers too. It's weirdly motivating to see your life as a grid.Suddenly every week is a binary view.
